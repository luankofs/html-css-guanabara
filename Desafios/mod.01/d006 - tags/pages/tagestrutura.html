Alguns códigos adicionais:

  • target="_blank" - abre nova aba
  • target="_self" - abre na mesma aba
  • rel="external" - indica para o google que é uma página externa
  • rel="next" - indica para o google que é minha proxima página
  • rel="prev" - indica para o google que é minha página anterior
  • rel="author" - indica link para site do autor do artigo atual
  • href lang - indica para o google o idioma da página
  • type="valor" (ul) - disc (bolinha preenchina) circle (bolinha sem preenchimento) square (quadrado preenchido
  • type="valor" (ol) - 1, a, A, I, i
  • source:media:type - adaptar imagens dinâmicas (tamanhos). Onde o source trabalha com o max width.
  • <a>download="" type="" - usa-se para download de arquivos.
  • picture - permite criar vários sources
  • audio - permite adicionar áudio - Tags adicionais: controls, type, loop, autoplay
  • video - permite adicionar vídeo - mesmo comando dos áudio.
  • poster - permite adicionar thumbnail no vídeo hospedagem própria.
